📋 **Les 7 thèmes à explorer de façon conversationnelle** :

🔍 **THÈME 1 - Connaissance de l'espace membre** :
- Question d'ouverture : "Avez-vous déjà exploré votre espace membre ?"
- Si NON → "Qu'est-ce qui vous a empêché d'y aller ? Manque de temps, d'information ?"
- Si OUI → "Qu'est-ce qui vous a le plus intéressé ? Y a-t-il des choses que vous n'avez pas trouvées ?"

🎯 **THÈME 2 - Expérience avec les services CCI** :
- Question d'ouverture : "Avez-vous déjà utilisé nos services ou participé à nos événements ?"
- Si NON → "Qu'est-ce qui vous freine ? Est-ce par manque d'information, de temps, ou autre chose ?"
- Si OUI → "Lesquels exactement ? Qu'est-ce qui vous a le plus apporté ?"

🚀 **THÈME 3 - Objectifs en Colombie** :
- Question d'ouverture : "Quel est votre principal défi ou objectif en Colombie actuellement ?"
- Toujours creuser → "Qu'est-ce qui vous motive le plus dans cette démarche ?"

⏰ **THÈME 4 - Projets à court terme** :
- Question d'ouverture : "Avez-vous des projets concrets prévus dans les 3 prochains mois ?"
- Si OUI → "Parlez-moi de ce projet, qu'est-ce qui vous enthousiasme le plus ?"
- Si NON → "Qu'est-ce qui vous empêche de vous lancer ? Quels sont les obstacles ?"

🤝 **THÈME 5 - Besoins d'accompagnement** :
- Question d'ouverture : "Sur quoi aimeriez-vous être accompagné par la CCI ?"
- Toujours approfondir → "Qu'est-ce qui serait le plus utile pour vous faire avancer ?"

💎 **THÈME 6 - Valeur perçue** :
- Question d'ouverture : "Qu'est-ce qui vous a apporté le plus de valeur dans votre relation avec la CCI ?"
- Si peu de valeur → "Qu'attendriez-vous de plus ? Qu'est-ce qui changerait la donne ?"

💡 **THÈME 7 - Suggestions d'amélioration** :
- Question d'ouverture : "Si vous pouviez ajouter un service à la CCI, ce serait quoi ?"
- Toujours creuser → "Qu'est-ce qui vous manque le plus actuellement ?"
